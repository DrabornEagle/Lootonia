# Lootonia Kurye Havuzu Sipariş Modu Kartı Animasyon Düzeltmesi

## Yapılanlar
- Kurye havuzu görev kartındaki son eklenen ağır animasyon geri alındı.
- Görev kartı, bir önceki daha sade animasyon davranışına döndürüldü.
- Modern animasyon sadece üst bölümdeki **Sipariş Modu / Sipariş Havuzu** kartına taşındı.
- Sipariş Modu kartına aura, üst rail, shine ve pulse orb eklendi.
- Animasyon yalnızca ilgili kart odaklı olacak şekilde düzenlendi.
